# Soul FM audio

`public/audio/isnt-she-lovely.mp3`

The Heaven turntable uses:

`public/audio/touching-god.mp3`

The Hell turntable uses:

`public/audio/really-love.mp3`

All three tracks are encoded at 128 kbps and are loaded only after playback is requested.

The shared player routes each source through its spectrum analyser and a room-specific reverb chain.
